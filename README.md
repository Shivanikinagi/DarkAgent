# DarkAgent
## ZK Compliance Layer for AI Agents

> Prove your agent followed the rules — without revealing what the rules are.

### The Problem
AI agents are taking consequential actions: executing trades, screening candidates,
provisioning infrastructure. When something goes wrong — or a regulator asks —
there is no cryptographic proof the agent acted within policy.
Screenshots don't count. Policy docs don't count.
Under the EU AI Act (full enforcement: August 2026), operational evidence is required.

### How DarkAgent Works
1. User sets policy rules, stored via ENS (alice.darkagent.eth)
2. AI agent submits action to DarkAgent proxy
3. Policy engine evaluates action against rules
4. **BLOCKED:** agent stopped, reason logged
5. **APPROVED:** Noir ZK circuit runs, proof submitted to Base Sepolia
6. On-chain proof: agent complied. Policy: hidden forever.

### What Makes This Different
| Tool            | Audit trail | Policy private? | On-chain proof? |
|-----------------|-------------|-----------------|-----------------|
| NeMo Guardrails | Yes         | No              | No              |
| Galileo         | Yes         | No              | No              |
| DarkAgent       | Yes         | **YES**         | **YES**         |

### Demo Scenarios
- **DeFi:** AI trading bot blocked from buying meme coins above spending limit
- **Hiring:** AI screening agent blocked from filtering on protected attributes

### Stack
Noir (Aztec) | Base Sepolia | Coinbase CDP | ENS | Vite + React + Tailwind

### Architecture
```
AI Agent generates an Action Payload
              |
              v
   +-----------------------+
   |  Action Interceptor   |  <-- proxy server, port 8787
   |  (Node.js / Express)  |
   +-----------------------+
              |
     Load policy from ENS
     (alice.darkagent.eth)
              |
   +-----------------------+
   |   Policy Engine       |  <-- evaluates JSON rules
   |   evaluatePolicy()    |
   +-----------------------+
         |         |
      BLOCK      APPROVE
         |         |
     Log +     Noir ZK Circuit
    notify     (compliance_check)
                   |
            Verifier.sol
            Base Sepolia
                   |
         On-chain proof: tx hash
         Policy: hidden forever
```

### Run Locally

```bash
# Terminal 1 — Start the proxy server
cd proxy && npm install && npm run dev

# Terminal 2 — Start the frontend
cd frontend && npm install && npm run dev

# Open http://localhost:5173
```
